# libgcc-runtime

## Summary

`libgcc-runtime` is the small GCC runtime shared object needed by the current
Rust/musl toolchain output for RootAsRole.

This is not a user-facing package. It exists so a RootAsRole binary that needs
`libgcc_s.so.1` can install through Moss without reaching for an unowned forge
library.

## System role

- Group: `libs`
- Installed plane: machine/system
- Why ONIX needs it: RootAsRole `dosr` and `chsr` currently contain a runtime
  `NEEDED` entry for `libgcc_s.so.1` when built by the Alpine/musl Rust
  toolchain in the forge.

## Implementation choice

- Implementation language: C/runtime assembly from GCC
- Rust alternative considered: not applicable for compiler runtime support
- Serious Rust implementation exists: `no`
- Selected implementation: GCC runtime `libgcc_s.so.1`
- Why this implementation: it is the runtime soname emitted by the current
  toolchain for the accepted RootAsRole build.

## Source and provenance

- Upstream: GCC runtime library
- Source archive or repository: bootstrap import from the Alpine/musl forge
  toolchain runtime
- Pinned version: detected by `make phase 511`
- Source hash: payload hash generated by `make phase 511`
- Patch set: none

This is an explicit bootstrap runtime-surface package. A later toolchain phase
should replace it with a source-built ONIX compiler-runtime package or eliminate
the dependency if the ONIX toolchain can do so cleanly.

## Build model

- Build environment: Alpine/musl forge VM
- Build tools: boulder, moss, readelf
- Target triple: forge-native musl target
- C runtime: `musl`
- Link model: dynamic-musl runtime provider
- Static attempt/result: RootAsRole was tried with `-static-libgcc`,
  `panic=abort`, and `+crt-static`; the first two kept `libgcc_s.so.1`, while
  `+crt-static` is not usable with this build.
- Shared runtime libraries: `libgcc_s.so.1`

Expected surface:

```text
/usr/lib/libgcc_s.so.1
/usr/lib/libgcc_s.so
```

## Runtime-clean contract

- No runtime `/nix/store` dependency: required
- No `/nix/store` shebangs: required
- No `/nix/store` RPATH/RUNPATH: required
- No glibc loader path: required
- No unexpected shared runtime libraries: required
- All expected shared libraries are ONIX-owned: required

## Runtime dependencies

Expected:

```text
libc.musl-x86_64.so.1   owner: musl
```

## Installed paths

Expected installed files:

```text
/usr/lib/libgcc_s.so.1
/usr/lib/libgcc_s.so
/usr/share/onix/packages/libgcc-runtime.md
```

## Exceptions

This is a minimal managed shared-library surface, accepted only because the
current RootAsRole build requires it.

No glibc and no `/nix/store` runtime paths are allowed.
